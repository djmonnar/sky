# 🌿 하늘땅 매장관리

진해 식당 "하늘땅"의 내부 운영용 웹앱 프로토타입입니다.
근무표 · 예약 · 근무기록 · 급여를 한 곳에서 관리합니다.

## 실행

```bash
npm install
npm run dev
```

## 기술 스택

- Vite + React 18 + TypeScript
- React Router (HashRouter)
- 순수 CSS 디자인 시스템 (`src/styles.css`)
- 목업 데이터 기반 프로토타입 (`src/data.ts`) — 백엔드 없이 동작

## 역할

- **실무자**: 오늘 근무 확인, 출/퇴근, 예약 확인, 근무기록 작성(버튼형 빠른 시간 입력), 전달사항
- **관리자**: 대시보드 KPI, 예약 등록/상태 변경, 주간 근무표 편집(프리셋/충돌 경고), 급여 승인/확정

사이드바 하단(데스크톱) 또는 우상단 아바타(모바일)로 역할을 전환할 수 있습니다.

## 화면 구성

| 경로 | 실무자 | 관리자 |
|---|---|---|
| `/` | 실무자 대시보드 | 관리자 대시보드 |
| `/reservations` | 오늘 예약 (조회/상태변경) | 예약 관리 (등록/수정) |
| `/schedule` | 내 근무표 | — |
| `/schedule-manage` | — | 근무표 관리 (주간 그리드 편집) |
| `/worklog` | 근무기록 작성 | — |
| `/payroll` | — | 급여 관리 (승인/확정) |
| `/notices` | 공지/전달사항 | 공지 등록 |

## 디자인 시스템

- **배경** 크림 `#f3efe3` / **포인트** 딥그린 `#2e3b2b`, `#3e5237`
- 라운드 카드(14px) + 은은한 그림자, 상태 배지 7종(예약확정/방문완료/취소/노쇼/단체/확인전화필요/예약대기)
- 데스크톱: 좌측 사이드바 + 메인 + 우측 상세 패널 / 모바일(≤720px): 하단 탭바 중심
- 시간 입력은 시계형 picker 대신 **버튼형 빠른 입력** (정시/10분 늦음/15분 연장, −15/+15분, 휴게 없음/30/60분)
